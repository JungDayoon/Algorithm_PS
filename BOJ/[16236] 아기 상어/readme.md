# [BOJ 16236] 아기 상어 - Python

### :computer: Algorithm

> BFS, 시뮬레이션



### :computer: Logic

`fishMap`: 상어와 물고기들의 위치를 나타내는 이차원 리스트

- **sequence**

  1. 상어의 위치에서 BFS를 진행하며 상어가 움직일 수 있는 좌표들을 queue에 넣는다.

  2. 한 번 큐를 다 돌고나면 새로 들어온 좌표들을 정렬하기 위해 

     ```python
     queue = sorted(queue)
     ```

     를 수행한다.

  3. 큐에 있는 상어들을 하나씩 꺼내면서 그 위치에 자신의 크기보다 작은 물고기가 있는 지 확인한다. 조건을 만족한다면, 그 물고기를 잡아먹고 상어의 위치를 그 자리로 옮겨준다.

  4. 잡아먹을 수 있는 물고기가 하나도 없을 때까지 반복한다.



### :computer: Review

--------

먹을 수 있는 물고기가 1마리보다 많다면, 거리가 가장 가까운 물고기를 먹으러 간다.

- 거리는 아기 상어가 있는 칸에서 물고기가 있는 칸으로 이동할 때, 지나야하는 칸의 개수의 최솟값이다.
- 거리가 가까운 물고기가 많다면, 가장 위에 있는 물고기, 그러한 물고기가 여러마리라면, 가장 왼쪽에 있는 물고기를 먹는다.
